# graphql-mern-apollo

> graphQL, react, apollo

## Quick Start

```bash
# Install dependencies (server & client)
npm run add

# Run server & client (:5000 & :3000)
npm run dev

# Server alone (:5000)
npm run server

# Client alone (:3000)
npm run client

# Graphiql - http://localhost:5000/
```
